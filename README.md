# Projet `vendeur` – Formulaire d'inscription vendeur

Ce projet est une petite application React (Vite) qui permet :
- d'enregistrer un vendeur,
- de choisir plusieurs catégories sous forme de bulles (véhicules, nourriture, électronique, vêtements),
- d'afficher les catégories sélectionnées en bas du formulaire avec possibilité de suppression,
- de rediriger vers une page de paiement.

## Installation

```bash
npm install
npm run dev
```

Puis ouvrez l'URL affichée (souvent http://localhost:5173).
